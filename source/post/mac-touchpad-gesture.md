### 个人常用手势

- 三指 => 拖动, 万物都可拖
- 三指轻触 => 查询
- 二指拖动 => 滚动
- 四指左右滑动 => 切换桌面(全屏)
- 二指轻触 => 右键菜单
- 一指放在触摸板上另外一指轻触 => 新窗口打开(Chrome)
- 双指左右滑 => 前进后退(Chrome)
