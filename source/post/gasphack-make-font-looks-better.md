[gasp_hack](http://pan.baidu.com/s/1eRRXhn0)

下载之后， 运行 `GaspHack_v2_MOD.bat` 等待补丁执行完毕， 然后到进入PE系统， 将 `.\workingDir\output`中的字体替换到系统字体目录，重启

重新执行一次ClearType调整，会发现字体看着瞬间就舒坦了

测试系统 Windows Server 2016
