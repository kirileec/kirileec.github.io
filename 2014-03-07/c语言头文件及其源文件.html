C中希望把一段代码，写成头文件的形式，作为后续调用，总不能每次都去粘贴代码吧

于是我按照规定写了这么两个文件

####getfilename.h

char *GetFileName(char *FilePath);

getfilename.c

#include <string.h>
char *GetFileName(char *FilePath)
{
    int len;
    char *current = NULL;
    len = strlen(FilePath);
    for (; len>0; len--)
    if (FilePath[len] == '134')
    {
        current = &FilePath[len + 1];
        break;
    }
    return current;
}
